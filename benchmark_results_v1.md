- **混合检索 vs 纯向量** — 忠实度（faithfulness）：下降 3.5%（0.6437 → 0.6214）
- **混合检索 vs 纯向量** — 答案相关性（answer_relevancy）：提升 13.0%（0.3045 → 0.3441）
- **混合检索 vs 纯向量** — 上下文精确度（context_precision）：下降 0.6%（0.4345 → 0.4321）
- **混合检索 vs 纯向量** — 上下文召回率（context_recall）：下降 6.0%（0.5675 → 0.5333）
- **Reranker 精排效果** — 忠实度（faithfulness）：下降 10.4%（0.6214 → 0.5569）
- **Reranker 精排效果** — 答案相关性（answer_relevancy）：下降 8.7%（0.3441 → 0.3143）
- **Reranker 精排效果** — 上下文精确度（context_precision）：下降 2.2%（0.4321 → 0.4226）
- **Reranker 精排效果** — 上下文召回率（context_recall）：下降 19.3%（0.5333 → 0.4302）
- **混合检索延迟**：P50 = 4124ms，满足交互体验要求（< 3s）
