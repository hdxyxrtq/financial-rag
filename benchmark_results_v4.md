- **混合检索 vs 纯向量** — 答案相关性（answer_relevancy）：提升 1.9%（0.3589 → 0.3658）
- **混合检索 vs 纯向量** — 上下文精确度（context_precision）：提升 28.8%（0.5693 → 0.7333）
- **混合检索 vs 纯向量** — 上下文召回率（context_recall）：下降 2.7%（0.8407 → 0.8178）
- **Reranker 精排效果** — 忠实度（faithfulness）：下降 6.0%（0.8692 → 0.8171）
- **Reranker 精排效果** — 答案相关性（answer_relevancy）：下降 3.5%（0.3658 → 0.3530）
- **Reranker 精排效果** — 上下文精确度（context_precision）：提升 2.0%（0.7333 → 0.7479）
- **Reranker 精排效果** — 上下文召回率（context_recall）：下降 6.0%（0.8178 → 0.7689）
- **混合检索延迟**：P50 = 2120ms，满足交互体验要求（< 3s）
